# agent-atlas

A from-zero learning roadmap for AI agents — curated resources organized progressively, with study notes (Chinese) published under the column **《The Agent Loop》**.

> Each issue: read a representative piece, then produce a comprehensive Chinese study note in `notes/` (with embedded original diagrams + a 小红书 cover). The roadmap below grows as issues accumulate.

## Repository layout

- `notes/` — Per-issue Chinese study notes, with embedded diagrams (`notes/images/`) and covers (`notes/covers/`).
- `scripts/` — Tooling. Currently: `gen-cover.sh` — a Gemini-CLI driven 小红书 cover generator with a locked unified style (warm cream / Anthropic terracotta / charcoal; 3:4 default).
- `CLAUDE.md` — Project rules and persistent memory (Chinese). Read this first if you want to understand how this repo is run.
- `PROGRESS.md` — Session-by-session progress log (Chinese, newest on top).
- `resources/` — Local-only original article backups (gitignored; copyright; for personal cross-referencing only).
